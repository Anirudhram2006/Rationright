# RationRight Project - Build & Deployment Instructions

## Overview
This project is a static website hosted on GitHub Pages. It uses HTML, CSS, JavaScript, and asset images stored in the `assets` folder.

---

## Folder Structure
- `index.html` — Main HTML file.
- `styles.css` — CSS styles for the site.
- `sw.js` — Service worker for caching and offline support.
- `assets/` — Folder containing images and icons used in the website.
